## DEPLOYMENT STEPS ON RENDER.COM

### 1. Go to Your Render Dashboard
```
https://render.com/dashboard
```

### 2. If You Already Have a Deployment

**Option A: Redeploy Existing Service**
1. Go to your service dashboard
2. Click "Manual Deploy" button (or "Redeploy latest commit")
3. Wait for deployment to complete (3-5 minutes)
4. Check logs for success

**Option B: Delete and Recreate (If Needed)**
1. Delete existing service
2. Follow "New Deployment" steps below

### 3. New Deployment (If Starting Fresh)

**Step 1: Create Web Service**
- Go to render.com
- Click "New +" 
- Select "Web Service"

**Step 2: Connect GitHub**
- Click "Connect account" (GitHub)
- Authorize Render
- Select repository: `harshvardhansingh903/SHL_ASSIGNMENT`
- Branch: `main`

**Step 3: Configure Service**
```
Name:                    shl-recommender
Environment:             Python 3
Region:                  (auto or choose)
Branch:                  main
Deployment Strategy:     Auto-deploy
```

**Step 4: Build & Start Commands**
```
Build Command (leave empty - uses requirements.txt)

OR

Build Command:
pip install -r requirements.txt

Start Command:
python -m uvicorn app.api_server:app --host 0.0.0.0 --port 8000
```

**Step 5: Environment Variables**
```
PYTHON_VERSION=3.11
```

**Step 6: Instance Type**
- Select: **Free** (or Starter if free tier full)

**Step 7: Click "Create Web Service"**
- Render will build and deploy automatically
- Takes 3-5 minutes first time
- Watch logs for any errors

## IF DEPLOYMENT STILL FAILS

### Check These Things:

1. **Logs Tab in Render**
   - Click service name
   - Select "Logs" 
   - Look for error message
   - Note the error

2. **Common Issues:**
   
   **Error: "No module named 'app'"**
   - Make sure __init__.py exists in app/
   - Check start command uses app.api_server:app
   
   **Error: "catalog not found"**
   - Verify data/shl_product_catalog_clean.json exists
   - Check GitHub repo has the file
   
   **Error: "requirements not found"**
   - Check requirements.txt is in root directory
   - Verify it has all dependencies

3. **To Fix and Redeploy:**
   ```bash
   # From your local machine:
   git push origin master:main
   
   # Then in Render dashboard:
   Click "Manual Deploy" or "Redeploy latest commit"
   ```
